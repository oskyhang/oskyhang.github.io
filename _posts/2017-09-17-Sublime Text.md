## Markdown插件
markwown preview

*install packate* 中搜索*markwown preview*

快捷键
*ctrl + b* : 生成html

添加快捷键
在preference->key binding -> user setting
```
[{
	"keys":["alt+m"], "command": "markdown_preview", "args": { "target": "browser"} 
}]
```

*alt+m* 可已在浏览器中预览编辑的md文件
